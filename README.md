### darkmode
```js
javascript:fetch("https://cdn.jsdelivr.net/gh/RuralAnemone/bookmarklets/public/darkmode.js").then((data=>data.text())).catch((e=>alert(e))).then((text=>eval(text))).catch((e=>alert(e)));
```
